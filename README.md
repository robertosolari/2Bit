# 2Bit — Binary Conversion Game

A web-based game to practice converting numbers between binary and decimal.

## How to Play

1. A number appears on screen (binary or decimal)
2. Type the converted value in the input field
3. Press **Enter** or click **OK** to submit
4. Answer correctly before the timer runs out

Correct answers increase your level and score. Wrong answers or timeouts reveal the correct answer.

## Features

- **Three modes**: Binary → Decimal, Decimal → Binary, or Mixed
- **Progressive difficulty**: numbers get larger as your level increases
- **Streak tracking**: 5 dots show your consecutive correct answers
- **Timer with progress bar**: 10 seconds per question
- **Bilingual**: English and Italian

## Tech Stack

HTML, CSS, JavaScript — no dependencies, no build step.

## Run Locally

Open `index.html` in a browser.

## Deploy

Static site — works on GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any static hosting.
